# Advent of Code 2025

Ruby solutions for [Advent of Code 2025](https://adventofcode.com/2025).

## Installation

Install the gem dependencies:

```bash
bundle install
```

## Usage

Create your daily solution files in the `lib/` directory and corresponding tests in the `spec/` directory.

Example structure for Day 1:
- `lib/day1.rb` - Solution implementation
- `spec/day1_spec.rb` - Tests for Day 1

## Running Tests

Run all tests:
```bash
bundle exec rspec
```

Run a specific test:
```bash
bundle exec rspec spec/day1_spec.rb
```
